# ユーザー管理画面 使い方ガイド

本ドキュメントでは、管理者向けに「ユーザー管理」画面の操作方法や仕様について解説します。

---

## 1. 従業員を新規登録する

管理者は、新しい従業員や管理者アカウントをシステムに登録することができます。

### 登録手順
1. 画面右上の **「+ ユーザー登録」** ボタンをクリックします。
2. 表示されるモーダルで、以下の情報を入力します。
   - **表示名 (必須)**: 画面上での表示や簡易的な確認に利用されます（最大50文字）。
   - **氏名 (必須)**: 給与申請等に利用するため、フルネーム（本名）を入力してください（最大100文字）。
   - **メールアドレス (必須)**: シフト申請や通知等に利用するメールアドレスを入力します。
   - **ロール (必須)**: アカウントの権限を選択します。
     - `従業員`: 一般の打刻・シフト参照権限
     - `管理者`: ユーザー管理・打刻端末制限管理などの全管理権限
3. **「保存」** ボタンをクリックすると登録が完了します。

> [!IMPORTANT]
> **アカウントIDに関する仕様**
> ユーザー登録画面には「アカウントID」の入力項目はありません。入力された **メールアドレス** が自動的に「アカウントID」として設定され、システム内での一意の識別子となります。

---

## 2. 従業員にNFCカード（FeliCa IDm）を登録する

打刻を行うためのNFCカード（交通系ICカード、社員証など）をユーザーに紐付けます。

### 事前準備
- **動作環境**: NFCカードの登録および打刻機能（WebUSB機能）は、**Google Chrome** または **Microsoft Edge** などの WebUSB 対応ブラウザでのみ動作します。Safari や Firefox では動作しません。
- **機器**: パソコンに Sony 製 NFCリーダー「PaSoRi (RC-S380 / RC-S300 など)」を接続しておきます。

### 登録手順
1. カードを登録したいユーザーの行にある **「カード」** ボタンをクリックします。
2. 「NFCカード管理」モーダルが開きます。
3. モーダル下部の「カードを新規登録」セクションで、**「接続」** ボタンをクリックします。
4. ブラウザのポップアップが表示されるので、接続されている **「PaSoRi」**（または `RC-S380` 等）を選択して **「接続」** をクリックします。
5. 接続に成功すると、ステータスが **「カードをかざしてください」** に変わります。
6. PaSoRi の上に登録したいカードをかざします。
7. 読み取りに成功すると、16桁の英数字（IDm）が表示されます。
8. 任意で **カード名**（例: 「通勤用Suica」「社員証」など）を入力します。
9. **「このカードを登録」** ボタンをクリックして登録を完了します。

> [!TIP]
> - 1人のユーザーに対して複数のカードを登録することができます。
> - 登録済みのカードは、モーダル上部の一覧から「名前変更」や「削除」を行うことができます。

## 3. ユーザーの削除（非有効化）と完全削除

ユーザーの登録解除には「削除（非有効化）」と「完全削除」の2つの段階があります。

### 削除（非有効化 / 論理削除）
- **操作**: ユーザー一覧で「有効」状態のユーザーの右側にある **「削除」** ボタンをクリックします。
- **意味**: アカウントが「無効」状態になります。無効化されたユーザーは、システムへのログインやカードによる打刻ができなくなります。
- **データの保持**: ユーザーのこれまでの出退勤履歴、登録済みのカード情報、シフトデータなどは **データベース内にすべて保持されます**。
- **復元**: 後から「編集」ボタンを押し、「有効」のチェックボックスをオンにして保存することで、いつでも元の状態に戻す（有効化する）ことができます。

### 完全削除（物理削除）
- **操作**: すでに「無効」状態になっているユーザーの右側にある **「完全削除」** ボタンをクリックします。
- **意味**: ユーザー情報をデータベースから完全に抹消します。
- **影響**: **この操作は取り消せません。** 対象ユーザーの出退勤履歴、シフト、紐付けられているカード情報など、すべての関連データが完全に削除されます。
- **制限**: システムの監査ログや整合性維持のため、内部専用の **`system` ユーザー** に対する「削除」および「完全削除」は行えないよう制限されています（UI上にボタンが表示されません）。

## 4. その他の便利機能とトラブルシューティング

### データのバックアップと復元
- **一括保存 (JSON)**: 現在登録されているすべてのユーザー情報を JSON ファイルとしてPCにエクスポートします。定期的なバックアップや、別環境への移行に利用できます。
- **一括復元**: エクスポートされた JSON ファイルを読み込んでユーザー情報を一括インポートします。
  - 同一のアカウントID（メールアドレス）を持つユーザーが既に存在する場合は、情報が上書き（更新）されます。
  - 存在しない場合は、新規ユーザーとして登録されます。

### ユーザーの検索
- ユーザー一覧上部の検索バーに「名前」「氏名」「メールアドレス」のいずれかを入力することで、該当するユーザーをリアルタイムで絞り込むことができます。

### トラブルシューティング：カードが読み取れない
1. **ブラウザの確認**: Safari や Firefox を使用していないか確認してください。Google Chrome または Microsoft Edge を使用してください。
2. **接続の確認**: USBハブなどを経由している場合、電力不足で動作しないことがあります。PCのUSBポートに直接接続し直してください。
3. **他のアプリの競合**: PC上で他のFeliCa読み取りソフト（SFCard Viewer等）が起動している場合、ブラウザからデバイスを占有できないことがあります。他の関連ソフトを終了してから再度お試しください。
